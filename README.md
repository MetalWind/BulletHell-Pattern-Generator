# BulletHell Pattern Generator для Unity

Данный проект предоставляет возможность для простого и максимально гибкого процесса создания паттернов для игр жанра Bullet Hell.

![ezgif com-optimize](https://github.com/MetalWind/BulletHell-Pattern-Generator/assets/150790210/a2c4a86c-6f9e-4a83-8c66-4723a18950ba)


### Необходимый минимум

Для создания подавляющего большинства узоров вам необходимо знать о трех вещах: классах Pattern, Projectile и Generator.

**Pattern** - это просто набор направленных точек. Он формируется при помощи нескольких параметров, отдельные точки нельзя добавлять или редактировать вручную. При описании генераторов, станет понятно, почему это сделано именно так. При желании, это можно легко обойти просто добавив в Generator несколько паттернов с нулевой задержкой и необходимым смещение, формируя узор из отдельных линий, кривых, фигур, или (тут уже при огромном желании) точек.

**Projectile** - MonoBehavior класс, который содержит в себе собственную модель, а так же набор модулей. Projectile активируется при вызове у него метода Launch, после чего он собирает все прикрепленные к объекту модули (наследники ProjectileModifierBase) и на каждом FixedUpdate применяет к себе эффекты каждого модуля. Большинство из них воздействует на модель, однако, ничто не мешает им воздействовать непосредственно на gameObject, как раз таки применяя данные из модели.

**Generator** - реализует потенциал снарядов и паттернов. Для работы генератор принимает на вход CompleteInstruction - класс содержащий в себе два паттерна, снаряд, логику перехода и количество залпов. На основе этих данных, CompleteInstruction способна генерировать промежуточные паттерны между двумя заданными, что позволяет создавать плавные, сложные и красивые узоры. Generator же просто применяет каждый полученный паттерн с указанной задержкой.

Всё остальное можно понять в процессе работы с этим инструментом, для начала, можно опробовать представленные примеры. Для этого просто откройте сцену с примером, и посмотрите, как все работает, думаю, вы быстро разберетесь.
